## StreetFood Webpage
This is a streetfood webpage showing the different food menus and descriptions.It contains different background colours and designs for creativity.
## Folder Structure
Main Page.html - This is the main page.
StreetFood.css - This is the css file,the editing file.
Readme.md - This is a short description/summary of the Folder.
Assets-20251117T175252Z-1-001 - This is a Folder containing all the images used in the page.
